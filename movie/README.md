# React 를 활용한 영화 사이트 만들기


## 사용한 도구 및 라이브러리
- styled-components
- react-router-dom
- react-icons
- axios
- redux
- react-redux
- redux-thunk
- swiper
```
$ yarn add styled-components
$ yarn add react-router-dom
$ yarn add react-icons
$ yarn add axios
$ yarn add redux
$ yarn add react-reudx
$ yarn add redux-thunk
$ yarn add swiper
```
